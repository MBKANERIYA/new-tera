# Teralease
A modern React web application built with Vite.

## Tech Stack
- Frontend: React.js
- Build Tool: Vite
- Language: JavaScript

## Directory Structure
- `frontend/` - Frontend React application (Vite)
- `backend/` - Backend API and services
- `knowledge-base/` - Project documentation and rules

## Reading Order
| File | Description |
|---|---|
| `README.md` | Core project information and rules |
| `changelog.md` | History of changes |

## Critical Rules
- Development Server: `cd frontend && npm run dev`
- Build Command: `cd frontend && npm run build`
- Preview Command: `cd frontend && npm run preview`
